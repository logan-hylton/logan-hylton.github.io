Project 1, Part 1: Wheel of Fortune

Overview

You will be creating a game similar to wheel of fortune, that will allow the player to guess a word, letter by letter. The difference is that there will be no spinning the wheel, so you can think of this game as hangman without the possibility of dying. We will have a part 2 to this project that will allow you to set difficulty levels and have a limited number of guesses; but, for now, we are just going to let the player guess until they get it right.

Instructions

You will be given a zip file containing two files: WheelOfFortune.java and GameRunner.java. I have written the basic game loop for you in GameRunner.java but it will be good for you to look over it and make sure you understand what is happening (it is very short). Your task is to implement the TakeTurn() method in the WheelOfFortune class, which will handle all of the logic relating to the game.

Here are a few (not all) of the things you will need to do in your takeTurn() method:

  • Prompt the user for input. This can be done by calling the getGuess() method that I provided.
  • Check to see if the letter that the player guessed appears in the answer array. If it does, then update answerBlanks array by replacing each occurence of "_", where the letter appears, with the letter that was guessed. If it doesn't, then add it to the list of incorrect guesses (guessedList).
  • Check to see if the word has been guessed. If it has, then end the game and tell the user that they won.
  • After each guess, if it wasn't the winning guess, you should print the state of the game (which letters have been guessed and which ones the user has gotten correct)
